# Teen Depression Prediction ML

## Project Overview
This project aims to develop a machine learning model that predicts the likelihood of teen depression based on various factors. The goal is to provide insights and support for mental health professionals, educators, and parents.

## Features
- **Data Analysis**: In-depth analysis of factors influencing teen depression.
- **Model Training**: Implementation of various machine learning algorithms to find the best predictor.
- **Interactive UI**: A user-friendly interface for inputting data and receiving predictions.
- **Visualization**: Graphical representations of data and prediction results.

## Requirements
- Python 3.8 or higher
- Libraries: pandas, numpy, scikit-learn, matplotlib, Flask
- A machine with an internet connection for downloading datasets and libraries.

## Usage Instructions
1. Clone the repository: `git clone https://github.com/Shahid742/teen-depression-prediction-ml`
2. Navigate to the project directory: `cd teen-depression-prediction-ml`
3. Install required libraries: `pip install -r requirements.txt`
4. Run the application: `python app.py`
5. Follow the prompts in the application.

## Model Details
- **Models Used**: Logistic Regression, Decision Trees, Random Forest, etc.
- **Evaluation Metrics**: Accuracy, Precision, Recall, F1 Score.
- The model is trained on a dataset containing demographic and psychological factors affecting teens.

## Results
The developed model achieved an accuracy of over 85% in predicting the likelihood of depression based on test data.
